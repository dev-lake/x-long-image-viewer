# Chrome Web Store Listing

## Product Details

Extension name:
X Long Image Viewer

Short description:
View long images on X/Twitter with full-page scrolling, zoom, and drag-to-pan controls.

Category:
Productivity

Language:
English

## Detailed Description

X Long Image Viewer improves how long images open on X/Twitter.

When you click an image from x.com or twitter.com, the extension opens the original media file in a focused full-page viewer. It is designed for comics, screenshots, infographics, chat captures, and other tall images that are hard to inspect in the default media preview.

Features:
- Opens X/Twitter media images at original quality when available.
- Fits long images to the viewport width for natural vertical reading.
- Supports mouse wheel scrolling, drag-to-pan, zoom in, zoom out, and reset.
- Supports keyboard controls: Esc to close, + and - to zoom, 0 to reset, arrow keys to nudge.
- Runs only on x.com and twitter.com.
- No account access, no tracking, no analytics, and no remote service.

## Store Form Answers

Single purpose:
This extension provides an enhanced viewer for long images on X/Twitter, making tall media easier to read with scrolling, zooming, and panning controls.

Permission justification:
The extension requests host access for https://x.com/* and https://twitter.com/* so its content script can detect clicked media images and open the enhanced viewer on those sites. It does not request Chrome API permissions.

Remote code:
The extension does not use remote code. All JavaScript and CSS are bundled in the extension package.

Data usage:
The extension does not collect, store, sell, or transmit user data. It reads clicked image URLs from X/Twitter pages only to display the selected image in the local viewer.

Privacy policy summary:
X Long Image Viewer does not collect personal information, browsing history, authentication data, or page content. It runs locally in the browser and does not send data to external servers.
